# Frontend Practice

This repository is a collection of small projects I built while learning HTML, CSS, and JavaScript. Each folder is a separate practice exercise with its own `index.html` and, in some cases, supporting CSS and JavaScript files.

## Projects

- Calorie Counter
- Cat Painting
- City Skyline
- Ferris Wheel
- Magazine
- Piano Designing
- Role Playing Game
- Sample Form
- Sample Table

## How to Use

1. Open the folder for the project you want to view.
2. Open the `index.html` file in a browser.
3. If you use VS Code, the Live Server extension also works well for previewing the pages.

## Notes

These projects are practice exercises, so some folders are focused only on layout or styling while others include more interactivity.
